# NICE DNB & NUMBLE <비재무 데이터를 활용한 중소기업 휴폐업 예측 챌린지

### 제공된 데이터
* 액티브 중소법인 재무보유: (28,982개 데이터, 28개 변수)
* 휴폐업 중소법인 재무보유: (6,739개 데이터, 28개 변수)
* 재무데이터: (109,142개 데이터, 76개 변수)

### EDA 및 전처리
* 결측치 제거
* 라벨링(휴폐업: 0, 액티브: 1)
* 모든 변수 7가지 구간으로 Binning(이상치 처리 및 모델링을 위한 전처리)
* 설립일자와 상장일자 업력으로 변경
* 파생변수 생성(총 24개)
